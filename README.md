# simple slider

カスタマイズが容易なタイマー付きスライダーです

## 機能

* 次へボタン(#slider_next)
* ページ移動/現在地ボタン(#slider_select li)
* 一定時間でスライド

## 使い方

こんな感じのhtmlで動作します

```
<div id="slider_wrapper">
    <ul class="slider">
        <li>slide1</li>
        <li>slide2</li>
        <li>slide3</li>
        <li>slide4</li>
    </ul>
    <div class="slider_next">next</div>
    <ul class="slider_select">
        <li>●</li>
        <li>●</li>
        <li>●</li>
        <li>●</li>
    </ul>
</div>
```
